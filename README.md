# Standoff-2-Opr-Price-Script
Standoff 2 Opr Price Script For Perfect Click

## Description
Opr Price script - Script for catching skins for a certain price (you set the price yourself).

## Features
- Telegram bots support
- Fast notifications
- Quick response to price changes


## Installation
- Download the latest version of the script from Play Market.
- Create a new script in Perfect Click.
- Paste the script code.
- Customise the script to your coordinates using the pipette.
- Add your Telegram bot and a chat id to which notifications will be sent.
- Run the script and enjoy catching them.


## Requirements
- Perfect Click 9.09+
